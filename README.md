# SHURA. Staff App

Beauty salon staff management app for SHURA (Glebe, Surry Hills, Ultimo Academy).

## Features
- 📅 Schedule & shift management
- 📢 Notices & service manual
- ⭐ Staff evaluation & leaderboard
- 📦 Inventory management (per location + consolidated)
- 💬 Client reviews & retention
- 💰 Wage summary (Jenny only)
- 🌐 4 languages: EN / 中文 / Tiếng Việt / 한국어

## Deploy to Vercel (Free)

1. Go to [vercel.com](https://vercel.com)
2. Sign up with your GitHub account (shura-room)
3. Click "Add New Project"
4. Select this repository
5. Click "Deploy" — done!

Your app will be live at: `https://shura-room.vercel.app`

## Files
- `index.html` — Main app (all-in-one)
- `manifest.json` — PWA (installable on phone)
- `vercel.json` — Deployment config
